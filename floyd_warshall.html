Floyd-Warshall

Programmation Dynamique sur les Graphes

Temps: $O(S^3)$ Espace: $O(S^2)$

Le Principe

L'algorithme de Floyd-Warshall permet de trouver les plus courts chemins entre toutes les paires de sommets d'un graphe (contrairement à Dijkstra qui part d'une source unique). Il a aussi l'avantage de fonctionner avec des poids négatifs !

L'idée repose sur une construction progressive astucieuse (la programmation dynamique) :

  • Étape initiale : On relève les distances directes entre chaque sommet $i$ et $j$ via la matrice d'adjacence. Si aucune arête ne les relie, la distance est considérée comme infinie ($\infty$).
  • Étape itérative : Pour chaque sommet du graphe qu'on va appeler $k$ (de $0$ à $S-1$), on passe en revue toutes les paires de sommets $(i, j)$ et on se pose une question simple :
    "Est-il plus rapide d'aller de $i$ à $j$ en gardant le chemin actuel, ou en faisant un détour par $k$ ?"

Si la distance de $i \rightarrow k$ additionnée à la distance de $k \rightarrow j$ est inférieure à la distance de $i \rightarrow j$ connue jusqu'à présent, on met à jour notre matrice des distances.

Implémentation

floyd_warshall.c 
void floyd_warshall(int graph[V][V]) {
    int dist[V][V];
    
    // 1. Initialisation
    for (int i = 0; i < V; i++)
        for (int j = 0; j < V; j++)
            dist[i][j] = graph[i][j];

    // 2. Programmation Dynamique
    for (int k = 0; k < V; k++) {
        for (int i = 0; i < V; i++) {
            for (int j = 0; j < V; j++) {
                // Si passer par k est plus court
                if (dist[i][k] + dist[k][j] < dist[i][j]) {
                    dist[i][j] = dist[i][k] + dist[k][j];
                }
            }
        }
    }
}
let floyd_warshall graph =
  let v = Array.length graph in
  let dist = Array.make_matrix v v 0 in

  (* 1. Initialisation *)
  for i = 0 to v - 1 do
    for j = 0 to v - 1 do
      dist.(i).(j) <- graph.(i).(j)
    done
  done;

  (* 2. Programmation Dynamique *)
  for k = 0 to v - 1 do
    for i = 0 to v - 1 do
      for j = 0 to v - 1 do
        if dist.(i).(k) + dist.(k).(j) < dist.(i).(j) then
          dist.(i).(j) <- dist.(i).(k) + dist.(k).(j)
      done
    done
  done;
  dist
